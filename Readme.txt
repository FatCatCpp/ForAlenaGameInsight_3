Программа проверки работоспособности всевозможных каналов данных ЭВМ на радиолокационной станции.
Протоколы: ethernet, оптоволоконный канал, последовательные порты (RS-232, RS-485).

Проводится циклический обмен пакетами данных по всем каналам: отправка определенного пакета, прием, распарсивание--->вывод о работоспособности канала.
